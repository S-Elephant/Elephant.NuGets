# About

ModelStateExtensions class that returns all the errors from the specified modelStateDictionary as a string.

# Installation

Choose one:

## **Package Manager** (Visual Studio GUI)
1. Right-click your project → "Manage NuGet Packages".
2. Search for `Elephant.Validation`.
3. Click "Install".

## **.NET CLI** (Command Line)
```bash
dotnet add package Elephant.Validation
```

## **PackageReference** (Project File)
```xml
<PackageReference Include="Elephant.Validation" Version="x.x.x" />
```

## **Package Manager (CLI)**
```bash
nuget install Elephant.Validation
```
