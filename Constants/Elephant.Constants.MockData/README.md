# About

Contains static ReadOnlyCollection unique mock data in x10, x100 and x1000 for:

- Cities (x10 and x100).
- Guids (x10 and x100).
- Postal Codes (Dutch 4PP and 6PP formats).
- Streets.
- Zip Codes (United States).
